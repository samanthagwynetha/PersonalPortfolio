# 👩‍💻 Samantha Arsua | Personal Portfolio

![Portfolio Preview](https://raw.githubusercontent.com/samanthagwynetha/PersonalPortfolio/main/assets/img/photo.jpeg)

A premium, minimalist personal portfolio designed to showcase my journey as a **Full-Stack Developer** and **UI/UX Designer**. This project reflects my focus on clean code, modern design aesthetics, and user-centered digital solutions.

🔗 **Live Demo:** [samantharsua.dev](https://samanthagwynetha.vercel.app/)

---

## ✨ Features

- **Modern Aesthetic:** Built with a "dark mode" first approach, featuring glassmorphism effects and high-end typography.
- **Dynamic Project Filtering:** Interactive gallery that allows users to filter by "Websites", "Apps", and "UI Design".
- **Responsive Design:** Fully optimized for all screen sizes, from mobile to ultra-wide monitors.
- **Smooth Animations:** Utilizes **Lottie** animations and custom scroll-reveal effects for an engaging user experience.
- **Interactive Tech Stack:** Visual representation of my core skills in Frontend, Backend, and Design Tools.
- **Contact Integration:** Functional contact form powered by **Formspree**.

## 🚀 Local Setup

To run this project locally:

1. **Clone the repository:**
   ```bash
   git clone https://github.com/samanthagwynetha/PersonalPortfolio.git
   ```
2. **Navigate to the directory:**
   ```bash
   cd PersonalPortfolio
   ```
3. **Open `index.html`:**
   Simply open the `index.html` file in your preferred browser, or use a "Live Server" extension in VS Code for real-time updates.
